# 🚀 Tecboard (Desenvolvida durante o curso da Alura)

Landing page desenvolvida para praticar conceitos de HTML e CSS.

## 🛠️ Tecnologias utilizadas
- HTML
- CSS3

## 📱 Responsividade
O projeto foi desenvolvido para funcionar em:
- Desktop
- Tablet
- Mobile

## 🎯 Objetivo
Este projeto foi criado com o objetivo de praticar:
- Estruturação de páginas com HTML
- Estilização com CSS
- Organização de layout

## 📚 Aprendizados
Durante o desenvolvimento, aprendi:
- Como criar uma landing page moderna
- Trabalhar com alinhamento e espaçamento
- Aplicar boas práticas de layout
- Adaptar o site para diferentes telas

## 🔧 Melhorias futuras
- Adicionar interatividade com JavaScript
- Implementar animações
- Criar novas seções na página
